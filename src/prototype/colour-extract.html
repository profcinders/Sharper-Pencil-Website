Colour Extraction - Prototype

A few years ago I made a form in C# that allowed users to upload a picture, present its main colours, let them choose one of these, and match that colour to the client's range of paints. Here I'm doing some of the same things, but in JS - you can upload an image, and the code will pull the main colours out of it using the image-palette library. Just for fun I'm also matching those colours to their closest Tailwind CSS colours. You can check out the source code on GitHub.
