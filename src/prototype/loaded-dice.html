Loaded Dice - Prototype

This is an experiment to see how randomised dice can be loaded to favour particular rolls, inspired by a particularly bad run of luck in a tabletop RPG game. You can check the source code on GitHub, and feel free to message me at contact@sharper-pencil.co.uk with your experience using this tool!
